# e3 config examples (archived)

Stock E3Next sample configs, moved here out of the live install so they don't clutter the real
per-character configs but stay available as **reference examples**. These are vendor samples for
**other servers** (Lazarus, PEQTGC) — not our `Sim_EQ` characters.

- `e3 Bot Inis/` — example per-character bot configs (full class loadouts: Bard, Cleric, Warrior, …,
  plus named samples like `Alara_Lazarus.ini`). Good reference when authoring a new Lieutenant/bot
  config: copy the closest class example and adapt.
- `e3 Macro Inis/Samples/` — sample Loot / Saved Groups / Tribute settings.

Live configs stay in the install at `tools/E3Next/config/` (gitignored except this `config/` subtree).
To use an example, copy it into `tools/E3Next/config/e3 Bot Inis/` and rename to your character +
server (e.g. `Oram_Sim_EQ.ini`), then tune.
